# Shop app

Learning project about a web shop, including frontend in Angular (using PrimeNG, Angular Material and Leaflet libraries), backend in Java Spring, database in MongoDB (products and shop related entities) and database in PostgreSQL (users data). Project contains Dockerfile for easy deployment and Postman collection for backend api testing.


## Docker deployment

Clone the project

```bash
  git clone https://github.com/krysztok/Shop-app.git
```

Go to the project directory

```bash
  cd Shop-app
```

Deploy in docker

```bash
  docker-compose up -d --build
```

Open client in web browser

```bash
  http://localhost:4200/
```


## Features

Main page:
- Navigation bar (from PrimeNG library) with search
- Menu (from PrimeNG library) showing all product categories with subcategories
- Recently seen products in chronological order
- Space for similar components like new products, outlet products and special offers

Categories:
- All categories can have picture
- Component showing subcategories of selected category

Product list:
- Navigation component that alows to go back to main category and main page
- Product filters of type:
    - Price filter (from - to)
    - Ratings (shows chosen rating and above)
    - Subcategories filter (if list shows main category products)
    - "String" filter (autogenerated filter, looks for all parameter options in product category)
    - "Integer" filter (autogenerated filter, looks for the biggest int number and generates options from 0 to max)
    - "Boolean" filter (true-false filter)
- Clear filter/filters option
- Sorting options that alows to sort by:
    - Rating (High-Low)
    - Name (A-Z and Z-A)
    - Price (Low-High and High-Low)
- Product compare (up to 3 products)
- Adding/removing product to/from cart and wish list

Search:
- Search input in navigation bar that looks for products with requested string in product name
- Basic product list features (filters and sorting)

Products:
- Navigation component that alows to go back to main category, subcategory and main page
- Adding/removing product to/from cart and wish list
- Rating option
- Showing photos, description, parameters and ratings

Cart:
- Double storage system (for local and logged user)
- Notification and removal for deleted products
- Notification and moving to wish list for unavailable Products
- Number of products in cart showed on navigation bar
- Autofill of client data if user is logged
- Components to chose transport option and payment 
- Making orders (with form validators)

Wish List:
- Double storage system (for local and logged user)
- Notification and removal for deleted products
- Number of products in wish list showed on navigation bar
- Adding/removing product to/from cart

Order status:
- Searching order by id
- Showing order details with status history
- Showing orders of user if logged

Contact:
- Map with marked shops (from Leaflet library, requires api-key to work)
- Components showing shops details

Account:
- Secured with guards on frontend and spring-security on backend (jwt tokens)
- Login and register options (with form validators)
- Changing user data/adress/password
- Cart and wish list storaged in database


Admin Panel:
- Secured with guards on frontend and spring-security on backend (jwt tokens)
- All inputs with validators
- Categories
    - List all categories with filters and sorting
    - Add new category
    - Edit category
    - Add/show/change/delete category image
    - Delete category

- Products
    - List all products with filters and sorting
    - Add new product
    - Edit product
    - Add/show/delete product images
    - Delete/deactivate/activate product
    - Show/delete product ratings

- Filters
    - List all product filters with filters and sorting
    - Add new filter (autogenerating filter options)
    - Check product related filters and add/remove options on product changes
    - Delete filter

- Orders
    - List all orders with filters and sorting
    - View order details
    - Change order status

- Shops
    - List all shops with filters and sorting
    - Add new shop
    - Edit shop
    - Delete/deactivate/activate shop

- Clients
    - List all clients with filters and sorting
    - View client details
    - Delete/deactivate/activate client account
